# Selenium

* 免費使用 (Open source project)
* 自動化控制使用者介面
* 支援多種瀏覽器：Firefox、Chrome、IE、Edga...
* 支援多平台：MAC、Window、Linux
* 支援多語言：Java、Python、Ruby、C#、Javascipt、C++

### Selenium 目前有四個主要的專案

* Selenium WebDriver
* Selenium Remote Control
* Selenium IDE
* Selenium Grid

### 優點

* 支援重複執行測試程式
* 支援並行執行
* 支援背景執行
* 提高測試準確定，減少人為產生的錯誤。
* 省下測試的時間

### 應用領域

* 自動化測試
* 網站擷取 / 網路爬蟲
