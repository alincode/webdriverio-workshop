# WebdriverIO 實戰練習 - 登入登出

**簡易測試規格**

測試項目 | 測試描述 | 預期結果
---------|----------|---------
 登入失敗 | 1. 輸入帳號 | 顯示登入失敗訊息
  | 2. 輸入錯誤密碼 | 
  | 3. 按送出按鈕 | 
  | 4. 檢查是否出現警告訊息 | 
  | 5. 警告訊息的文字內容，是否如預期 | 
 登入成功 | 1. 輸入帳號 | 進入首頁
  | 2. 輸入正確密碼 | 
  | 3. 按送出按鈕 | 
  | 4. 檢查是否存在登出連結 | 
 登出 | 1. 點選登出 | 顯示登出成功訊息
  | 2. 檢查是否出現登出成功的訊息 | 

**建立第一個前端測試程式**

```
// 建立空的 test/specs 資料夾
mkdir -p ./test/specs

// 先把 Selenium Server 執行起來 (視窗一)
webdriver-manage start

// 然後再跑測試程式 (視窗二)
npm test
```
